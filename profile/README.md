# Noma Travel

Noma is a corporate travel management platform powered by AI agents. It helps companies search, plan, book, and manage business trips — flights, hotels, transfers, and ground transport — through a conversational interface backed by a modular, extensible backend.


## Stack

**Backend:** Python 3.12 · Flask · AWS Lambda · DynamoDB · S3 · Cognito · API Gateway

**Frontend:** Next.js · React · TypeScript · Tailwind CSS · shadcn/ui

**AI:** OpenAI LLM integration via a custom interpret/act agent loop
